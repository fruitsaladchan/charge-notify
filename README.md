# charge-notify

## how to use

### Install a notifaction dameon like dunst

```
yay -S dunst
```

### Make the script executable

```
-chmod +x the script
```
### start the script on startup through your window manager
```
- for i3wm

exec --no-startup ~/Path/To/Script

- for hyprland

exec-once = ~/Path/To/Script

```
